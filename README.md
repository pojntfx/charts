# Felicitas Pojtinger's Helm Charts

Helm chart repository for my projects.

## Installation

```bash
$ helm repo add pojntfx https://pojntfx.github.io/charts/
```

## Usage

```bash
$ helm search repo pojntfx
$ helm install pojntfx/<chart>
```
